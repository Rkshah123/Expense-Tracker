# 💰 Expense Tracker

A simple and responsive Expense Tracker application built using **HTML, Bootstrap 5, and JavaScript**. The application allows users to add, edit, and delete expense records while storing data in the browser's Local Storage.

## 🚀 Features

- Add new expenses
- Edit existing expenses
- Delete expenses
- Store data using Local Storage
- Responsive Bootstrap UI
- No backend required
- Fast and lightweight

## 📸 Preview

The Expense Tracker helps users manage their daily spending by recording:

- Expense Amount
- Description
- Category

All expense records are automatically saved in the browser and remain available after page refresh.

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6)
- Local Storage API

## 📖 Functionality

### Add Expense

Enter:

- Expense Amount
- Description
- Category

Then click **Add Expense**.

### Edit Expense

Click the **Edit** button to modify an existing expense record.

### Delete Expense

Click the **Delete** button to remove an expense.

### Data Persistence

All expenses are stored in the browser using **Local Storage**, ensuring data remains available even after refreshing or reopening the page.
